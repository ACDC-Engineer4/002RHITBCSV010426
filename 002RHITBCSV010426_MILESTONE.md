<h1>111RIMSTSVV010326 MILESTONE </h1>

## Milestone 1 – Requirement Analysis and System Architecture
- Review and finalize input/output specifications (24V to 12V DC).

- Identify protection requirements (fuse, reverse polarity using bridge rectifier).


## Milestone 2 – Circuit Design and Component Selection
- Design buck converter schematic.
- Calculate and select component values for stable 12V output.
- Select fuse rating based on current requirements(SRT 1A 300V). 
- Design reverse polarity protection using bridge rectifier.


## Milestone 3 – PCB Layout Design
- Design single-layer PCB layout as per requirement.
- Optimize component placement for thermal and electrical performance.
- Ensure proper routing for high-current paths.


## Milestone 4 – Documentation 
- Prepare final schematic and PCB documentation.
- Provide BOM (Bill of Materials).
- Include design notes and protection feature explanation.


## Milestone 5 – Prototype Testing and Validation 
- Assemble prototype PCB.
- Test input (24V) to output (12V) performance.
- Verify protection features (fuse operation, reverse polarity behavior).
